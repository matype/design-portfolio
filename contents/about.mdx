### Bio

1990年8月30日、和歌山県生まれ。妻と娘と犬 (チワワ) と暮らしています。

2015年に就職を機に上京し、2020年10月に和歌山に戻ってきました。
現在はフルリモートで仕事をしています。コロナ以前の2016年から部分的にリモートワークをしていたので慣れており、在宅環境も整えています。

FOSS とタイポグラフィーとゲームが好きです。
自作の FOSS フォントを GitHub で公開しています。
また、ゲームは最近はもっぱらエルデンリングナイトレインをしています。
その場その場で最適な行動を選択し、その期待値で勝負をするようなリアルタイムストラテジー系のゲームが好きです。

仕事では、デザイナーのいないプロジェクトや組織に、1人目のデザイナーとしての参加がほとんどです。プロダクトの UI デザインを中心に、ユーザーリサーチやモデリング、ブランディングのための VI / CI の設計、マーケティング用のグラフィック制作、デザイナー採用のサポートなど、一気通貫することが多いです。
意匠よりも機能性とパフォーマンスを、トレンドよりも普遍性を重視する傾向があります。

客観視したときの自分の強みは、システムのアーキテクチャを理解して、エンジニアが話す言葉でコミュニケーションができるところです。
特に Web フロントエンドの領域は、一時個人的に研究していたこともあり、実装レベルの知識と実務経験があります。
情報セキュリティと統計的なデータ分析にも (デザイナー比較では) 明るいです。
